# Tactical Warzone V7.0 - AWM Special Edition

ブラウザで動作する、Three.jsベースのタクティカル3D FPSゲームエンジンです。
最新のアップデートでは、AWMスナイパーライフルのモデリング刷新と、全武器に対応したダイナミックUIが追加されました。

## 🚀 主な特徴

- **本格的な射撃システム**: ADS（覗き込み）、反動アニメーション、薬莢（シェル）の排出エフェクト。
- **AWM Special Update**:
    - **巨大化モデル**: 1.4倍にスケールアップされた迫力ある銃体。
    - **中空スコープ**: 内部をくり抜いたチューブ構造により、ADS時にレンズ越しに敵を狙う没入感を実現。
    - **極細レティクル**: 視認性を損なわない精密な十字線。
    - **自動フロントサイト・カリング**: ADS中、視界を遮るフロントサイトを自動で非表示化。
- **高度なUI/UX**:
    - **円形ロード進捗バー**: レティクル周辺にリロードや発射間隔（コッキング）を可視化するゲージを表示。
    - **ミニマップ**: 敵の位置をリアルタイムで把握可能。
    - **モバイル最適化**: ジョイスティックと専用ボタンによる直感的なタッチ操作。
- **ゲームモード**:
    - デプロイメニューから「昼間」と「夜間」を選択可能。
    - 敵の数を指定する「ミッションモード」と、無限に敵が湧く「エンドレスモード」を搭載。

## 🛠 アップデート詳細 (V7.0)

### AWM スコープの改善
- **支柱の侵食修正**: スコープを支える2つのマウントがチューブ内部に突き抜けないよう設計を変更。
- **ポジション調整**: 銃をより手前に構えるよう配置し、スコープの視認性を向上。
- **線幅の微調整**: 十字線の太さを極限まで細くし、遠距離のターゲットを正確に捉えられるようにしました。

### バグ修正
- **薬莢排出ロジック**: 全ての武器において、薬莢がカメラの向きに関わらず「右斜め後ろ」へ正確に飛び出すよう物理計算を修正。

## 🎮 操作方法

- **移動**: 画面左側のジョイスティック
- **視点移動**: 画面右側をスワイプ
- **FIRE**: 射撃
- **ADS**: スコープを覗く / 腰だめ
- **RELOAD**: 弾薬の補充
- **SWAP**: 武器の切り替え（AR / SMG / AWM / HG）
- **JUMP**: ジャンプ

## 🔧 技術スタック

- **Engine**: Three.js (WebGL)
- **Language**: JavaScript (ES6+), HTML5, CSS3
- **Font**: Share Tech Mono (Google Fonts)

## 📦 セットアップ

1. `index.html` をブラウザで開くだけでプレイ可能です。
2. 特殊なサーバー設定は不要ですが、安定したフレームレートを維持するために最新のブラウザ（Chrome/Safari推奨）を使用してください。
